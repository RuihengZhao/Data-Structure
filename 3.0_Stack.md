# Stack

Implementation:
- using array: saved memory but not dynamic

```
class Stack {
    static final int MAX = 1000; // fixed size
    int top;
    int a[] = new int[MAX];

    Stack() {
        top = -1;
    }

    boolean isEmpty() {
        return (top < 0);
    }

    boolean push(int x) {
        if (top >= MAX - 1) {
            System.out.println("Stack Overflow");
            return false;
        } else {
            a[++top] = x;
            System.out.println(x + " pushed into stack");
            return true;
        }
    }

    int pop() {
        if (top < 0) {
            System.out.println("Stack Underflow");
            return 0;
        } else {
            return a[top--];
        }
    }
}
```

- using linked list: extra memory but dynamic

```
class Stack {
    Node top;

    class Node {
        int data;
        Node next;

        Node(int d) {
            data = d;
        }
    }

    boolean isEmpty() {
        return (top == null);
    }

    boolean push(int x) {
        Node new_node = new Node(x);
        new_node.next = top;
        top = new_node;
    }

    int pop() {
        if (top == null) {
            System.out.println("Stack Underflow");
            return 0;
        } else {
            int x = top.data;
            top = top.next;
            return x;
        }
    }
}
```
